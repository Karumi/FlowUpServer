Getting started with Flow Up

Add the FlowUp Android SDK to your build:

Add the following dependency to your build.gradle file: 


dependencies {

    compile 'io.flowup:android-sdk:@sdkVersion'

    testCompile 'io.flowup:android-sdk-no-op:@sdkVersion'

}

Initialize FlowUp:

Initialize the FlowUp Android SDK in the your Application class onCreate() method. 


FlowUp.Builder.with(this)

    .apiKey("@apiKey.getValue")

    .forceReports(BuildConfig.DEBUG)

    .start();

Build and run your application:

Build and run your app in an emulator or device to start seeing data. Remember to build the app using the DEBUG build type and review adjust the device date/timezone properly.

Refresh this page and start seeing data in just a few seconds:

Within seconds you will see data in your FlowUp Dashboards.

If your project uses lint remember to add this configuration to your project as follows.


android {
...
lintOptions {
    abortOnError true
    disable 'InvalidPackage'
    }
}
