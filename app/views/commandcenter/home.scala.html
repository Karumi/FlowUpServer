Get started with Flow Up

Add FlowUp to your build

Merge the following into your build.gradle file:

dependencies {
compile 'io.flowup:android-sdk:<LAST_VERSION_RELEASED>'
testCompile 'io.flowup:android-sdk-no-op:<LAST_VERSION_RELEASED>'
}

Initialize FlowUp

In your default Activity import the FlowUp class.

import io.flowup.FlowUp;

And in the onCreate() method add this call to initialize FlowUp.

FlowUp.Builder.with(this)
.apiKey("@apiKey.value")
.forceReports(BuildConfig.DEBUG)
.logEnabled(BuildConfig.DEBUG)
.start();

Build and run your application

Clean your project and then run your app in an emulator or device to start seeing data.

Start seeing data in just a few minutes

Within minutes you will see data in your FlowUp Dashboards.
