Getting started with Flow Up in iOS

Add the SDK to your build

Add the following dependency to your Podfile: 

pod 'FlowUpIOSSDK', '~> @sdkVersion'

Initialize FlowUp

Initialize the FlowUp iOS SDK in your AppDelegate class application:didFinishLaunchingWithOptions: method.

Objective-C:

[FlowUp application:application
didFinishLaunchingWithOptions:launchOptions
                 apiKey:@"@apiKey.getValue"
     isDebugModeEnabled:YES];

Swift:

FlowUp.application(application: application,
                   didFinishLaunchingWithOptions: launchOptions,
                   apiKey: "@apiKey.getValue",
                   isDebugModeEnabled: true)

Build and run your application

Build and run your app in the simulator or real device to start seeing data.

Refresh this page and start seeing data in just a few seconds

Within seconds you will see data in your FlowUp Dashboards.

Additionally, remember you can send an email to support@@flowup.io asking for any doubt or issue.
